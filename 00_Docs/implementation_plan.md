# プロジェクト実装計画書: NDB_XXX_perioperative_oral_care
**テーマ**: 癌周術期における医科歯科連携の地域格差解析

## 1. 目的
主要がん手術（医科）が行われている症例に対し、どの程度の割合で周術期口腔機能管理（歯科）が実施されているかを都道府県単位で定量化し、その地域格差と要因（歯科資源量）を明らかにする。

## 2. 解析指標の定義
- **分母 (Denominator)**: 主要がん手術（Kコード）の算定回数（都道府県別）
    - 食道癌、胃癌、肺癌、大腸癌、頭頸部癌等の主要手術を抽出・合算。
- **分子 (Numerator)**: 周術期等口腔機能管理料（歯科）の算定回数（都道府県別）
    - `B001-2`（Ⅰ）および `B001-3`（Ⅱ）を合算する。
- **主要アウトカム**: 周術期口腔ケア実施率 (%)

## 3. 具体的な抽出対象コード（暫定）
### 医科（手術：Kコード）
- 食道: K529, K529-2, K529-3
- 胃: K655, K655-2, K655-4, K655-5, K657, K657-2
- 肺: K514, K514-2
- 大腸: K719, K719-2, K719-3, K740, K740-2
- 頭頸部: K457, K458, K459

### 歯科（管理料：Bコード）
- 周術期等口腔機能管理料（Ⅰ）：B001-2
- 周術期等口腔機能管理料（Ⅱ）：B001-3

## 4. 解析ステップ
1.  **データ抽出**: 第10回NDBオープンデータの「医科診療行為」および「歯科診療行為」の都道府県別Excelから対象コードを抽出。
2.  **データ結合**: 都道府県をキーとして、医科（手術件数）と歯科（管理料件数）を結合。
3.  **統計解析**: 実施率の算出と、地域リソース（歯科医師数等）との相関分析。
